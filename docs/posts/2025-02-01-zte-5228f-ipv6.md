# 中兴5228F解决IPV6问题

最近配了IPV6，发现中兴5228F上的设备无法获取IPV6地址，网上搜刮了两条指令，解决IPV6无法使用问题。

```
ZXR10(config)#ipv6 nd-snooping link-local enable

ZXR10(config)#ipv6 nd-snooping global enable
```

配置完记得输入 `write` 保存
