# 用 VitePress + Tailwind 搭一个更适合 AI 的博客

## 为什么不是 NotionNext

NotionNext 更像内容同步方案，不像源码优先方案。

## 为什么是 VitePress

因为它对 Markdown 和 Git 工作流更友好，也更容易让 AI 直接修改文件。

## Tailwind 在这里的作用

它不负责"提供现成博客系统"，它负责让页面更容易做出你想要的视觉风格。
